# Excel Analytics Platform (MERN Stack)

Tech Stack: React.js, Node.js, Express.js, MongoDB, Tailwind CSS

Developed a full-stack analytics platform that allows users to upload Excel files, generate 2D/3D charts, and view AI-generated insights.

Implemented features like user authentication, admin dashboard, upload history, and chart export (PDF/PNG).

Gained strong hands-on experience in integrating backend APIs, data visualization, and AI assistance.
